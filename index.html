Cascading Style Sheet

Syntax 

                P{
                    color:blue;
                    font-size:12px;     
                }

                * How to save CSS files?
                        fileName.css

How To Add CSS in HTML? 

                * There are three ways to add CSS in HTML.
                      1.External CSS
                      2.Internal CSS
                      3.Inline CSS
                
                * First priority given to the Inline CSS then to both External & Internal CSS equally,
                  In External & Internal the one written in last will be given more priority after Inline.
